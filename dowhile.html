Estrutura de Repetição: {Loop Do While}

DEFINIÇÃO

O loop do while é usado para repetir um bloco de código pelo menos uma vez e continuar repetindo enquanto uma condição especifica for verdadeira. Exemplo em javascript abaixo:

FORMA GERAL

do{

ㅤoque vai fazer

}while (condiçao_para_fazer);

EXEMPLO

const readline = require('readline-sync');

let x = 0

console.log("Bem Vindo Ao Programa!")

do{

ㅤ if(x == 1){

ㅤㅤㅤㅤbreak;

ㅤㅤ} else{

ㅤㅤㅤㅤconsole.log("X diferente de 1!")

ㅤㅤ }

}while (x!=1)

console.log("Programa encerrado")

COMO ISSO FUNCIONA ?

Primeiro o usuario entra com x

O programa confere se X = 1

Se for verdadeiro o programa para ali

Senao for, ele confere o valor de X novamente com o While(x!=1)

X foi declarado como 0, então ele vai repetir

O programa só para quando x = 1, já que vai atender a condiçao do break
