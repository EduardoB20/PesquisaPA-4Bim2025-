Switch Case

DEFINIÇÃO

O loop do while é usado para repetir um bloco de código enquanto uma condição especifica for falsa ou verdadeira. Exemplo em javascript abaixo:

FORMA GERAL

switch(expressão){

ㅤㅤcase valor1:

ㅤㅤㅤㅤ//Instruções executadas quando o resultado da expressão for igual á valor1

ㅤㅤㅤㅤbreak;

ㅤㅤcase valor2:

ㅤㅤㅤㅤ//Instruções executadas quando o resultado da expressão for igual á valor2

ㅤㅤㅤㅤbreak;

ㅤㅤcase valorN:

ㅤㅤㅤㅤ//Instruções executadas quando o resultado da expressão for igual á valorN

ㅤㅤㅤㅤbreak;

ㅤㅤdefault:

ㅤㅤㅤ//Instruções executadas quando o resultado da expressão for igual a nenhum valor dos cases

ㅤㅤㅤㅤbreak;

}

EXEMPLO

const readline = require('readline-sync');

console.log("Bem Vindo Ao Programa!")

let fruta = parseInt(readline.question("Digite 1 para comprar Laranja; Digite 2 para comprar Pera; Digite 3 para nao comprar nada\n"))

switch(fruta){

ㅤㅤcase 1:

ㅤㅤㅤㅤconsole.log ("Voce comprou laranjas")

ㅤㅤㅤㅤbreak;

ㅤㅤcase 2:

ㅤㅤㅤㅤconsole.log("Voce comprou peras")

ㅤㅤㅤㅤbreak;

ㅤㅤcase 3:

ㅤㅤㅤㅤconsole.log("Voce não comprou nada")

ㅤㅤㅤㅤbreak;

ㅤㅤdefault:

ㅤㅤㅤㅤconsole.log("Hmm, parece que esse item nao esta listado")

ㅤㅤㅤㅤbreak;

}

console.log("Programa encerrado")

EXECUTE O CÓDIGO

Bem Vindo Ao Programa!

Digite 1 para comprar Laranja; Digite 2 para comprar Pera; Digite 3 para nao comprar nada

COMO ISSO FUNCIONA ?

Primeiro o usuario entra com o número da fruta

Depois chamamos o switch

Ele vai fazer varias checagens com o valor da variavel 'fruta'

A checagem ocorre por meio do 'case'

É como se o programa estivesse dizendo:

caso for (valor_especifico) faz tudo oque esta em BAIXO

O break aqui serve para ele parar naquele lugar.

Se nao houvesse o break ele iria apenas continuar até o fim, ou até um outro break.

Default é o padrao, ou seja, se não for nenhum dos cases(casos), ele vai fazer aquilo.

O que acontece se esquecer um break?

Se esquecer um break então o script irá rodar até um dos pontos a seguir: Um outro break. O fim do switch.

EXEMPLO

const readline = require('readline-sync');

console.log("Bem Vindo Ao Programa!")

let Numero = parseInt(readline.question("Digite o número"))

switch(Numero){

ㅤㅤcase 1:

ㅤㅤㅤㅤconsole.log ("1 é Ímpar")

ㅤㅤㅤㅤbreak;

ㅤㅤcase 2:

ㅤㅤㅤㅤconsole.log("2 é par")

ㅤㅤㅤㅤ// COMENTARIO: Deveria existir um break aqui!

ㅤㅤcase 3:

ㅤㅤㅤㅤconsole.log("3 é Ímpar")

ㅤㅤㅤㅤ// já que nao existe ele continua até o proximo break(esse aqui)

ㅤㅤㅤㅤbreak;

ㅤㅤdefault:

ㅤㅤㅤㅤconsole.log("Não sei que número é esse!")

ㅤㅤㅤㅤbreak;

}

console.log("Programa encerrado")
